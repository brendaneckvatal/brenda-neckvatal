# Brenda Neckvatal Leadership Library

This repository is a public collection of leadership writing, frameworks, scripts, and behavioral definitions focused on navigating difficult people, authority challenges, and high-pressure workplace dynamics.

## What You’ll Find Here

### Articles
Real-world leadership analysis covering toxic behavior, authority erosion, gossip, coaching resistance, and power dynamics.

### Frameworks
Structured leadership models designed to stabilize conversations, reinforce accountability, and protect authority.

### Scripts
Practical language leaders can use in high-pressure moments to redirect conversations without escalation.

### Definitions
Plain-language explanations of behavioral and leadership concepts to support shared understanding.

## Purpose

This library exists to help leaders maintain clarity, authority, and disciplined communication when navigating difficult workplace situations.
