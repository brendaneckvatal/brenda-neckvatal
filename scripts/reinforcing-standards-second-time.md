# Reinforcing Standards the Second Time

## Purpose

Provide language leaders can use when behavior repeats and credibility is at risk.

## Scenario

An employee repeats behavior that was previously addressed but not formally reinforced.

## Script Language

Leader:

“We addressed this before. Since it’s resurfaced, we’re reinforcing the standard moving forward.”

Optional follow-up:

“This isn’t escalation. It’s consistency.”

## Notes

Deliver calmly. The second time is instructional for the entire team.
